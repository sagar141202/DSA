# Gray Code

## Problem Statement
The Gray Code is a sequence of binary numbers where each number differs from the previous one by only one bit. Given an integer `n`, generate all possible Gray Code sequences of length `n`. For example, if `n = 2`, the output should be `[00, 01, 11, 10]`. The sequence should be in lexicographical order.

## Approach
The Gray Code sequence can be generated using recursion and backtracking. We start with an empty sequence and keep appending `0` and `1` to the previous sequence, making sure that each new sequence differs from the previous one by only one bit. The key insight is to use the previous sequence to generate the next sequence.

## Complexity
- Time: O(2^n)
- Space: O(2^n)

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<int> grayCode(int n) {
        vector<int> result;
        for (int i = 0; i < (1 << n); i++) {
            // Calculate the Gray Code using bitwise XOR operation
            int gray = i ^ (i >> 1);
            result.push_back(gray);
        }
        return result;
    }
};

// Alternatively, using recursion
class Solution {
public:
    vector<int> grayCode(int n) {
        vector<int> result;
        helper(n, result);
        return result;
    }

    void helper(int n, vector<int>& result) {
        if (n == 0) {
            result.push_back(0);
            return;
        }
        helper(n - 1, result);
        int size = result.size();
        for (int i = size - 1; i >= 0; i--) {
            result.push_back(result[i] | (1 << (n - 1)));
        }
    }
};
```

## Key Takeaways
- The Gray Code sequence has a recursive structure, where each new sequence is generated by appending `0` and `1` to the previous sequence.
- The bitwise XOR operation can be used to calculate the Gray Code from the binary representation of a number.
- The recursive approach can be more intuitive, but the iterative approach using bitwise operations is more efficient.
