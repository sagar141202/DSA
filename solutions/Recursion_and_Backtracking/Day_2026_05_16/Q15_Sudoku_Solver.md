# Sudoku Solver

## Problem Statement
Given a partially filled Sudoku board, the task is to fill in the missing numbers so that each row, column, and 3x3 sub-grid contains the numbers 1-9 without repeating any number. The input is a 2D vector representing the Sudoku board, where 0 represents an empty cell. The output should be the solved Sudoku board. For example, given the following board:
```
5 3 0 | 0 7 0 | 0 0 0
6 0 0 | 1 9 5 | 0 0 0
0 9 8 | 0 0 0 | 0 6 0
-------------------------
8 0 0 | 0 6 0 | 0 0 3
4 0 0 | 8 0 3 | 0 0 1
7 0 0 | 0 2 0 | 0 0 6
-------------------------
0 6 0 | 0 0 0 | 2 8 0
0 0 0 | 4 1 9 | 0 0 5
0 0 0 | 0 8 0 | 0 7 9
```
The output should be the solved Sudoku board.

## Approach
The approach to solve this problem is to use recursion and backtracking. We start by filling in the empty cells one by one, and for each cell, we try numbers from 1 to 9. If the number is valid, we recursively try to fill in the rest of the board. If we cannot fill in the rest of the board, we backtrack and try the next number.

## Complexity
- Time: O(9^(n*n)) where n is the size of the Sudoku board
- Space: O(n*n) for the recursion stack

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

bool isValid(vector<vector<int>>& board, int row, int col, int num) {
    // Check if the number already exists in the row or column
    for (int i = 0; i < 9; i++) {
        if (board[row][i] == num || board[i][col] == num) {
            return false;
        }
    }

    // Check if the number exists in the 3x3 sub-grid
    int startRow = row - row % 3;
    int startCol = col - col % 3;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            if (board[i + startRow][j + startCol] == num) {
                return false;
            }
        }
    }

    return true;
}

bool solveSudoku(vector<vector<int>>& board) {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (board[i][j] == 0) {
                for (int num = 1; num <= 9; num++) {
                    if (isValid(board, i, j, num)) {
                        board[i][j] = num;
                        if (solveSudoku(board)) {
                            return true;
                        }
                        board[i][j] = 0;
                    }
                }
                return false;
            }
        }
    }
    return true;
}

int main() {
    vector<vector<int>> board = {
        {5, 3, 0, 0, 7, 0, 0, 0, 0},
        {6, 0, 0, 1, 9, 5, 0, 0, 0},
        {0, 9, 8, 0, 0, 0, 0, 6, 0},
        {8, 0, 0, 0, 6, 0, 0, 0, 3},
        {4, 0, 0, 8, 0, 3, 0, 0, 1},
        {7, 0, 0, 0, 2, 0, 0, 0, 6},
        {0, 6, 0, 0, 0, 0, 2, 8, 0},
        {0, 0, 0, 4, 1, 9, 0, 0, 5},
        {0, 0, 0, 0, 8, 0, 0, 7, 9}
    };

    if (solveSudoku(board)) {
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                cout << board[i][j] << " ";
            }
            cout << endl;
        }
    } else {
        cout << "No solution exists" << endl;
    }

    return 0;
}
```

## Key Takeaways
- The use of recursion and backtracking can be an effective way to solve problems that involve trying different combinations of values.
- The `isValid` function is used to check if a number can be placed in a given cell, and it checks the row, column, and 3x3 sub-grid.
- The `solveSudoku` function uses recursion to try different numbers in each empty cell, and it backtracks when it finds a solution that does not work.
