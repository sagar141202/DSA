# Next Permutation

## Problem Statement
Given a collection of distinct integers, return all permutations of the collection. However, instead of returning all permutations, we will generate the next permutation in lexicographic order. If the current permutation is the last permutation in lexicographic order, we will return the first permutation. The input array will contain distinct integers, and we need to find the next permutation of the given array. For example, given the array [1, 2, 3], the next permutation would be [1, 3, 2]. If the input array is [3, 2, 1], the next permutation would be [1, 2, 3].

## Approach
The algorithm to find the next permutation involves finding the first decreasing element from the right, swapping it with the smallest element greater than it, and then reversing the elements to the right of the swapped element. This approach ensures that we get the next permutation in lexicographic order.

## Complexity
- Time: O(n)
- Space: O(1)

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

void nextPermutation(vector<int>& nums) {
    // Find the first decreasing element from the right
    int i = nums.size() - 2;
    while (i >= 0 && nums[i] >= nums[i + 1]) {
        i--;
    }
    
    // If the entire array is in descending order, reverse it to get the first permutation
    if (i >= 0) {
        int j = nums.size() - 1;
        while (nums[j] <= nums[i]) {
            j--;
        }
        swap(nums[i], nums[j]);
    }
    
    // Reverse the elements to the right of the swapped element
    int left = i + 1;
    int right = nums.size() - 1;
    while (left < right) {
        swap(nums[left], nums[right]);
        left++;
        right--;
    }
}

int main() {
    vector<int> nums = {1, 2, 3};
    nextPermutation(nums);
    for (int num : nums) {
        cout << num << " ";
    }
    return 0;
}
```

## Key Takeaways
- The next permutation in lexicographic order can be generated by finding the first decreasing element from the right and swapping it with the smallest element greater than it.
- The elements to the right of the swapped element need to be reversed to get the next permutation.
- If the entire array is in descending order, reversing it will give the first permutation.
