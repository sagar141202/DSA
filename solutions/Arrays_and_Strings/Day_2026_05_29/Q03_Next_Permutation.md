# Next Permutation

## Problem Statement
Given an array of integers, generate the next permutation in lexicographic order. If the given array is the last permutation in lexicographic order, return the first permutation. The array consists of distinct integers from 1 to n, where n is the length of the array. For example, given the array [1, 2, 3], the next permutation would be [1, 3, 2]. If the given array is [3, 2, 1], the next permutation would be [1, 2, 3].

## Approach
The algorithm finds the largest index k such that the current element is less than the next one, then finds the largest index l > k such that the current element at index k is less than the element at index l, and swaps them. Finally, it reverses the sub-array starting from index k + 1 to the end.

## Complexity
- Time: O(n)
- Space: O(1)

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

void nextPermutation(vector<int>& nums) {
    int n = nums.size();
    int i = n - 2;
    // find the largest index k such that nums[k] < nums[k + 1]
    while (i >= 0 && nums[i] >= nums[i + 1]) {
        i--;
    }
    if (i >= 0) {
        int j = n - 1;
        // find the largest index l > k such that nums[k] < nums[l]
        while (nums[j] <= nums[i]) {
            j--;
        }
        // swap nums[k] and nums[l]
        swap(nums[i], nums[j]);
    }
    // reverse the sub-array starting from index k + 1 to the end
    reverse(nums.begin() + i + 1, nums.end());
}

int main() {
    vector<int> nums = {1, 2, 3};
    nextPermutation(nums);
    for (int num : nums) {
        cout << num << " ";
    }
    return 0;
}
```

## Key Takeaways
- The next permutation of an array can be generated by finding the largest index k such that the current element is less than the next one.
- If no such index k exists, the array is the last permutation in lexicographic order, so we return the first permutation by reversing the entire array.
- The time complexity of the algorithm is O(n) because we only need to iterate through the array once to find the index k and then reverse the sub-array.
