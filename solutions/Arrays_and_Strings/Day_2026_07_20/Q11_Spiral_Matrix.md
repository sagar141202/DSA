# Spiral Matrix

## Problem Statement
Given an integer `n`, generate a spiral matrix of size `n x n` filled with numbers from 1 to `n * n` in a spiral pattern, starting from the top left and moving clockwise. The spiral pattern should start with 1 at the top left, then move right to `n`, then down to `2n`, then left to `3n-1`, and so on. For example, for `n = 3`, the spiral matrix should be:
```
1 2 3
8 9 4
7 6 5
```
The function should take an integer `n` as input and return a 2D vector representing the spiral matrix.

## Approach
The algorithm uses four pointers to track the current boundaries of the matrix, and fills in the numbers in a spiral pattern by iterating over the elements in a clockwise direction. The pointers are updated after each iteration to move to the next layer of the spiral.

## Complexity
- Time: O(n^2)
- Space: O(n^2)

## C++ Solution
```cpp
#include <vector>

class Solution {
public:
    std::vector<std::vector<int>> generateMatrix(int n) {
        std::vector<std::vector<int>> matrix(n, std::vector<int>(n, 0));
        int top = 0;
        int bottom = n - 1;
        int left = 0;
        int right = n - 1;
        int num = 1;
        
        while (top <= bottom && left <= right) {
            // Fill in the top row from left to right
            for (int i = left; i <= right; i++) {
                matrix[top][i] = num++;
            }
            top++;
            
            // Fill in the right column from top to bottom
            for (int i = top; i <= bottom; i++) {
                matrix[i][right] = num++;
            }
            right--;
            
            // Fill in the bottom row from right to left
            if (top <= bottom) {
                for (int i = right; i >= left; i--) {
                    matrix[bottom][i] = num++;
                }
                bottom--;
            }
            
            // Fill in the left column from bottom to top
            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    matrix[i][left] = num++;
                }
                left++;
            }
        }
        
        return matrix;
    }
};
```

## Key Takeaways
- The spiral pattern can be generated by iterating over the elements in a clockwise direction and updating the pointers after each iteration.
- The time complexity is O(n^2) because we need to fill in all the elements in the matrix.
- The space complexity is O(n^2) because we need to store the entire matrix.
