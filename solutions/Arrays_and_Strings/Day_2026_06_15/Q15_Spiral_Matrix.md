# Spiral Matrix

## Problem Statement
Given an m x n matrix, return all elements of the matrix in spiral order. The matrix is filled with distinct positive integers. The spiral order starts from the top-left corner and moves clockwise. For example, given the following matrix:
```
[
 [ 1, 2, 3 ],
 [ 4, 5, 6 ],
 [ 7, 8, 9 ]
]
```
The spiral order is: `[1, 2, 3, 6, 9, 8, 7, 4, 5]`. The function should take as input a 2D vector of integers and return a 1D vector of integers representing the spiral order of the input matrix.

## Approach
The algorithm uses four pointers to keep track of the current boundaries of the matrix. It iterates over the elements in a spiral order by moving the pointers inward after each iteration. The spiral order is generated by first traversing the top row, then the right column, the bottom row, and finally the left column.

## Complexity
- Time: O(m * n)
- Space: O(m * n)

## C++ Solution
```cpp
#include <vector>
using namespace std;

vector<int> spiralOrder(vector<vector<int>>& matrix) {
    vector<int> result;
    if (matrix.empty()) return result;
    
    int rowBegin = 0;
    int rowEnd = matrix.size() - 1;
    int colBegin = 0;
    int colEnd = matrix[0].size() - 1;
    
    while (rowBegin <= rowEnd && colBegin <= colEnd) {
        // Traverse from left to right
        for (int i = colBegin; i <= colEnd; i++) {
            result.push_back(matrix[rowBegin][i]);
        }
        rowBegin++;
        
        // Traverse from top to bottom
        for (int i = rowBegin; i <= rowEnd; i++) {
            result.push_back(matrix[i][colEnd]);
        }
        colEnd--;
        
        // Traverse from right to left
        if (rowBegin <= rowEnd) {
            for (int i = colEnd; i >= colBegin; i--) {
                result.push_back(matrix[rowEnd][i]);
            }
            rowEnd--;
        }
        
        // Traverse from bottom to top
        if (colBegin <= colEnd) {
            for (int i = rowEnd; i >= rowBegin; i--) {
                result.push_back(matrix[i][colBegin]);
            }
            colBegin++;
        }
    }
    
    return result;
}
```

## Key Takeaways
- Initialize four pointers to keep track of the current boundaries of the matrix.
- Traverse the matrix in a spiral order by moving the pointers inward after each iteration.
- Handle edge cases where the matrix is empty or has only one row or column.
