# Huffman Coding

## Problem Statement
Huffman coding is a variable-length prefix code that assigns shorter codes to more frequently occurring characters in a dataset. Given a string of characters, determine the Huffman codes for each character and calculate the total length of the encoded string. The string consists of lowercase English letters (a-z) and has a maximum length of 1000 characters. The frequency of each character is provided as input.

## Approach
The Huffman coding algorithm uses a priority queue to build a binary tree, where each node represents a character and its frequency. The algorithm repeatedly merges the two nodes with the lowest frequencies until only one node remains. The Huffman codes are then generated by traversing the binary tree.

## Complexity
- Time: O(n log n)
- Space: O(n)

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

// Define a structure to represent a node in the Huffman tree
struct Node {
    char character;
    int frequency;
    Node* left;
    Node* right;
};

// Compare two nodes based on their frequencies
struct compare {
    bool operator()(const Node* a, const Node* b) {
        return a->frequency > b->frequency;
    }
};

// Function to build the Huffman tree
Node* buildHuffmanTree(unordered_map<char, int>& frequency) {
    priority_queue<Node*, vector<Node*>, compare> pq;
    for (auto& it : frequency) {
        Node* node = new Node();
        node->character = it.first;
        node->frequency = it.second;
        node->left = nullptr;
        node->right = nullptr;
        pq.push(node);
    }

    while (pq.size() > 1) {
        Node* left = pq.top();
        pq.pop();
        Node* right = pq.top();
        pq.pop();

        Node* newNode = new Node();
        newNode->character = '\0';
        newNode->frequency = left->frequency + right->frequency;
        newNode->left = left;
        newNode->right = right;

        pq.push(newNode);
    }

    return pq.top();
}

// Function to generate the Huffman codes
void generateHuffmanCodes(Node* root, string code, unordered_map<char, string>& huffmanCodes) {
    if (root == nullptr) return;

    if (root->left == nullptr && root->right == nullptr) {
        huffmanCodes[root->character] = code;
    }

    generateHuffmanCodes(root->left, code + "0", huffmanCodes);
    generateHuffmanCodes(root->right, code + "1", huffmanCodes);
}

// Function to calculate the total length of the encoded string
int calculateEncodedLength(string str, unordered_map<char, string>& huffmanCodes) {
    int length = 0;
    for (char c : str) {
        length += huffmanCodes[c].length();
    }
    return length;
}

int main() {
    string str;
    cin >> str;

    unordered_map<char, int> frequency;
    for (char c : str) {
        frequency[c]++;
    }

    Node* root = buildHuffmanTree(frequency);
    unordered_map<char, string> huffmanCodes;
    generateHuffmanCodes(root, "", huffmanCodes);

    int encodedLength = calculateEncodedLength(str, huffmanCodes);
    cout << "Huffman Codes: " << endl;
    for (auto& it : huffmanCodes) {
        cout << it.first << ": " << it.second << endl;
    }
    cout << "Encoded Length: " << encodedLength << endl;

    return 0;
}
```

## Key Takeaways
- Huffman coding is a lossless data compression algorithm that assigns shorter codes to more frequently occurring characters.
- The Huffman tree is constructed using a priority queue, where each node represents a character and its frequency.
- The Huffman codes are generated by traversing the binary tree, and the total length of the encoded string is calculated by summing the lengths of the codes for each character.
