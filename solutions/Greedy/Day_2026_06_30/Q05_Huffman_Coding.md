# Huffman Coding

## Problem Statement
Huffman coding is a lossless data compression algorithm. The goal is to assign variable-length binary codes to input characters based on their frequencies. The problem requires finding a Huffman coding scheme that minimizes the total length of the encoded string. Given a string of characters, each with a frequency of occurrence, we need to construct a Huffman tree and generate the Huffman codes for each character.

## Approach
The approach involves building a priority queue of nodes, where each node represents a character and its frequency. We then repeatedly remove the two nodes with the lowest frequencies, combine them into a new node, and add it back to the queue. This process continues until only one node remains, which is the root of the Huffman tree.

## Complexity
- Time: O(n log n)
- Space: O(n)

## C++ Solution
```cpp
#include <bits/stdc++.h>
using namespace std;

// Structure to represent a node in the Huffman tree
struct Node {
    char character;
    int frequency;
    Node* left;
    Node* right;
};

// Comparison function for the priority queue
struct compare {
    bool operator()(const Node* a, const Node* b) {
        return a->frequency > b->frequency;
    }
};

// Function to build the Huffman tree
Node* buildHuffmanTree(vector<pair<char, int>>& frequencies) {
    priority_queue<Node*, vector<Node*>, compare> queue;
    
    // Create a leaf node for each character and add it to the queue
    for (auto& frequency : frequencies) {
        Node* node = new Node();
        node->character = frequency.first;
        node->frequency = frequency.second;
        node->left = nullptr;
        node->right = nullptr;
        queue.push(node);
    }
    
    // Build the Huffman tree
    while (queue.size() > 1) {
        Node* left = queue.top();
        queue.pop();
        Node* right = queue.top();
        queue.pop();
        
        Node* internalNode = new Node();
        internalNode->character = '\0';
        internalNode->frequency = left->frequency + right->frequency;
        internalNode->left = left;
        internalNode->right = right;
        
        queue.push(internalNode);
    }
    
    return queue.top();
}

// Function to generate the Huffman codes
void generateHuffmanCodes(Node* node, string code, unordered_map<char, string>& huffmanCodes) {
    if (node == nullptr) {
        return;
    }
    
    if (node->left == nullptr && node->right == nullptr) {
        huffmanCodes[node->character] = code;
    }
    
    generateHuffmanCodes(node->left, code + "0", huffmanCodes);
    generateHuffmanCodes(node->right, code + "1", huffmanCodes);
}

int main() {
    string input = "this is an example for huffman encoding";
    vector<pair<char, int>> frequencies;
    
    // Calculate the frequency of each character
    unordered_map<char, int> frequencyMap;
    for (char c : input) {
        frequencyMap[c]++;
    }
    
    for (auto& frequency : frequencyMap) {
        frequencies.push_back(frequency);
    }
    
    Node* root = buildHuffmanTree(frequencies);
    unordered_map<char, string> huffmanCodes;
    generateHuffmanCodes(root, "", huffmanCodes);
    
    // Print the Huffman codes
    for (auto& code : huffmanCodes) {
        cout << code.first << ": " << code.second << endl;
    }
    
    return 0;
}
```

## Key Takeaways
- Huffman coding is a variable-length prefix code, where no code is a prefix of another code.
- The Huffman tree is a full binary tree, where each leaf node represents a character and its frequency.
- The Huffman codes are generated by traversing the Huffman tree from the root to each leaf node, assigning a 0 to the left child and a 1 to the right child.
